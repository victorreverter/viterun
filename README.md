# ViteRun ⚡🏃‍♂️

The Marathoner's Mission Control.

ViteRun is a high-performance web dashboard designed by a marathoner, for marathoners. It bridges the gap between raw training data and actionable race-day strategy. Built with a focus on speed (Vite/Next.js) and precision, it provides the essential "Runner's Math" suite required to crush your PBs.

## Key Features (Phase 1 - Current)
- **Race Predictor**: Leveraging the Riegel Formula to accurately forecast marathon finishes.
- **Physiology Lab**: Instant HR Zone calculation (Karvonen Method) and Body Composition tracking.
- **Pace Logic**: Multi-unit pace and split calculators for interval sessions.
- **Persistence**: No-login required; your data stays with you via local browser storage.

## The Roadmap
- **Phase 2**: Strava/Garmin API integration, PostgreSQL backend for multi-year trend analysis, and interactive 16-week training plan generators.

## Tech Stack
- **Frontend**: React, Next.js, Tailwind CSS, Recharts.
- **Design**: Custom "High-Contrast Sports" Dark Theme.
- **Performance**: Optimized for mobile-first use at the track.
